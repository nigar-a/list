DOCUMENTATION


Reviews System (v2.1) plugin for LaraClassified.

Installation

To install this plugin:
  • Copy the plugin's (uncompressed) folder into the script's plugins folder like this: /app/Plugins/reviews/
  • Go to your Admin panel → Plugins (and click on "Install") to install it.

Compatibility

  • v2.1+ => LaraClassified v6.9+
  • v2.0 => LaraClassified v6.8
  • v1.9 => LaraClassified v6.6 to v6.7
  • v1.8 => LaraClassified v5.7 to v6.5
  • v1.7 => LaraClassified v5.2 to v5.6
  • v1.6 => LaraClassified v5.1
  • v1.3 to v1.5 => LaraClassified v4.4 to v5.0
  • v1.1 to v1.2 => LaraClassified v3.7 to v4.3
  • v1.0 => LaraClassified v3.1 to v3.6

Update

Replace all the plugin's files on your server.

Configuration

  • Restrict access to registered users from your Admin panel → Setup → General Settings → Reviews → "Allow Guests to post Reviews"

Translation

To translate this plugin into your language:
  • Go to the languages folder at /app/Plugins/reviews/resources/lang/
  • Duplicate the folder /app/Plugins/reviews/resources/lang/en/ to /app/Plugins/reviews/resources/lang/<abbr>/
    NOTE: Don't forget replace <abbr> by your language code (e.g. "de" for German language).
  • Open the file /app/Plugins/reviews/resources/lang/<abbr>/messages.php and translate it.
    NOTE: To translate the file, you must change the values and not the keys of PHP array.

Uninstallation

To uninstall this plugin, go to your Admin panel → Plugins (and click on "Uninstall") to uninstall it.

Removing

To remove this plugin:
  • Uninstall it
  • Go in the folder /app/Plugins/ and remove the folder reviews/.
